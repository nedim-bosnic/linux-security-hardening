## Dijagrami

> Preporuka: drži slike u podfolderu `images/` (PNG/SVG), a tekstove kao `.md` fajlove.

- `images/` — dijagrami (VLAN, DMZ, mikrosegmentacija, itd.)
- dodatni `.md` fajlovi — pojedinačne teme (npr. `dmz.md`, `vlan-segmentation.md`, `ztna.md`)

## Konvencija imenovanja

- Bosanski: `*-bs.md` ili bez sufiksa (npr. `network-segmentation.md`)
- Engleski: `*-eng.md` (npr. `network-segmentation-eng.md`)
- Slike: kratko i jasno (npr. `dmz.png`, `vlan-segmentation.svg`)
